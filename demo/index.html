About Rustoku UI

            
          

          

            

              Rustoku UI is a WebAssembly-powered interface for
              Rustoku, a lightning-fast Sudoku solver and
              generator built entirely in Rust.
            

            
Features

            
    
              
  • 
                Fast solving: Solves puzzles in microseconds
                using bitmasking and backtracking with MRV heuristic
              
    • 
              
  • 
                Puzzle generation: Creates unique puzzles with
                guaranteed single solutions
              
    • 
              
  • 
                Human techniques: Employs solving strategies
                like Naked Singles, Hidden Pairs, and more
              
    • 
              
  • 
                Multiple distributions: Available as Rust
                library (crate), CLI, WebAssembly, and Python bindings
